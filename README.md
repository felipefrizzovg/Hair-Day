<h2>🛠️ Installation Steps:</h2>

<p>1. Install packages</p>

```
npm install
```

<p>2. Run API</p>

```
npm run server
```

<p>3. Run Webpack</p>

```
npm run build
```

<p>4. Run Website</p>

```
npm run dev
```
